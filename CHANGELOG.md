# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/),
and this project adheres to [Semantic Versioning](https://semver.org/).

## [0.2.0] - 2026-05-19

### Added
- MiMo provider support for local model integration
- PyQt6 GUI with modern desktop interface
- Chat widget with message history
- Cost tracking widget with visual statistics
- Settings dialog for API key management
- Export functionality (PDF, Markdown, HTML)
- File handler for document processing (PDF, DOCX)
- Internationalization (i18n) support
- Web search integration
- 18 built-in prompt templates for various use cases
- Privacy protection with automatic PII detection and masking
- Three privacy levels (low, medium, high)

### Changed
- Improved project structure with src layout
- Enhanced CLI with more subcommands
- Updated dependency versions

## [0.1.0] - 2026-05-01

### Added
- Initial release
- Multi-provider support: OpenAI, Claude, Ollama
- Unified chat interface for all providers
- CLI tool with interactive mode
- Local SQLite storage for conversation history
- Basic cost tracking per API call
- YAML-based configuration system
- Streaming response support
- Environment variable support via .env files
